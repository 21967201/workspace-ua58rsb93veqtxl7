## 🔴 严重问题1: Gateway 服务未运行

### 检测结果
```
Runtime: stopped (ERROR: The system cannot find the file specified.)
Service: Scheduled Task (missing)
Service not installed. Run: openclaw gateway install
```

### 根本原因
1. OpenClaw Gateway 计划任务不存在
2. Windows 上 `openclaw gateway install` 命令被禁用（显示 "Nix mode detected; service install is disabled"）
3. Gateway 无法以服务方式稳定运行

### 影响 ⚠️
**这是导致"任务莫名其妙中断"的主要原因！**

Gateway 是 QClaw 的核心通信枢纽，未运行会导致：
- 任务调度失败
- 心跳检测失效  
- Agent 间通信中断
- Cron 任务无法执行

## ✅ 修复方案（请手动执行）

由于自动执行环境受限，请按以下步骤**手动执行修复**：

### 步骤1: 清理所有 QClaw 进程

**打开 PowerShell（管理员权限），执行**:

```powershell
# 1. 停止所有 QClaw 进程

# 2. 等待2秒

# 3. 验证清理结果（应该无输出）

```

**预期结果**: 最后一个命令无输出（所有进程已停止）

---

### 步骤2: 创建 Gateway 计划任务

**在 PowerShell（管理员权限）中执行**:

```powershell
# 1. 创建 Gateway 计划任务（开机自启）
$action = New-ScheduledTaskAction -Execute "D:\QClaw\v0.2.24.540\QClaw.exe" -Argument "gateway run"
$trigger = New-ScheduledTaskTrigger -AtStartup
$principal = New-ScheduledTaskPrincipal -UserId "Administrator" -LogonType ServiceAccount -RunLevel Highest
Register-ScheduledTask -TaskName "OpenClaw Gateway" -Action $action -Trigger $trigger -Principal $principal -Force

# 2. 验证任务创建成功
schtasks /Query /TN "OpenClaw Gateway"
```

**预期结果**: 显示任务信息，状态为 "Ready"

## 🔧 长期改进建议

### 1. 添加健康检查机制

创建定时任务，每5分钟检查 Gateway 状态：

**保存为** `D:\QClawX\scripts\qclaw-health-check.ps1`:

```powershell
# QClaw 健康检查脚本
$gatewayStatus = openclaw gateway status 2>&1
if ($gatewayStatus -match "stopped|error|ERROR") {
    $msg = "$(Get-Date): Gateway is down, attempting restart..."
    Write-EventLog -LogName Application -Source "QClaw" -EntryType Error -EventId 1001 -Message $msg -ErrorAction SilentlyContinue
    schtasks /Run /TN "OpenClaw Gateway"
}

# 检查进程数量
$processCount = (Get-Process QClaw -ErrorAction SilentlyContinue).Count
if ($processCount -gt 3) {
    $msg = "$(Get-Date): Detected $processCount QClaw processes, consider cleaning up"
    Write-EventLog -LogName Application -Source "QClaw" -EntryType Warning -EventId 1002 -Message $msg -ErrorAction SilentlyContinue
}
```

**创建计划任务**:
```powershell
$action = New-ScheduledTaskAction -Execute "PowerShell.exe" -Argument "-File D:\QClawX\scripts\qclaw-health-check.ps1"
$trigger = New-ScheduledTaskTrigger -Once -RepetitionInterval (New-TimeSpan -Minutes 5) -At (Get-Date)
Register-ScheduledTask -TaskName "QClaw Health Check" -Action $action -Trigger $trigger -Force
```

### 2. 进程管理脚本

在启动新 Agent 前检查是否已有实例运行：

**保存为** `D:\QClawX\scripts\qclaw-process-check.ps1`:

```powershell
$processes = Get-Process QClaw -ErrorAction SilentlyContinue
if ($processes.Count -gt 2) {
    Write-Output "警告: 检测到 $($processes.Count) 个 QClaw 进程，建议清理"
    $processes | Select-Object Name, Id, StartTime, @{Name="Memory(MB)";Expression={$_.WorkingSet / 1MB -as [int]}} | Format-Table
    
    # 可选：自动清理最老的进程（保留最新的2个）
    $processes | Sort-Object StartTime | Select-Object -First ($processes.Count - 2) | Stop-Process -Force
}
```
