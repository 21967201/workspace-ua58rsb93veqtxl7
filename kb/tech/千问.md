# 千问 (Qwen)

## 技术概述
千问 (Qwen) 是阿里巴巴集团开发的大语言模型系列，包括文本、多模态等多种版本。

## 核心特性
- **多模态能力**: 支持文本、图像、音频等多模态输入
- **多语言支持**: 中英文表现优异，支持多种语言
- **开源版本**: Qwen-7B, Qwen-14B 等开源模型
- **商业化**: 通过阿里云对外提供API服务

## 使用场景
- **企业应用**: 客户服务、内容生成、数据分析
- **开发者**: 通过阿里云API集成到应用
- **研究**: 学术界用于多语言、多模态研究

## 代码示例

### Python API 调用
```python
from dashscope import Generation

response = Generation.call(
    model='qwen-turbo',
    prompt='请介绍一下千问模型'
)
print(response.output.text)
```

## 最佳实践

1. **提示词工程**: 千问对中文提示词理解较好，建议使用清晰的中文指令
2. **多模态输入**: 充分利用千问的多模态能力，结合图像和文本
3. **温度参数**: 创意任务用较高温度 (0.8-1.0)，精确任务用较低温度 (0.1-0.3)

## 常见问题

### Q1: 千问和GPT有什么区别？
A: 千问在中文理解上有优势，且提供更多开源版本供本地部署。

### Q2: 如何获取API访问权限？
A: 通过阿里云官网申请，新用户有免费额度。
